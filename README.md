Programming challenge from Fox Studios by Greg Sandell, Dec 2025

Location of this repo: `main` branch of [https://github.com/gregsandell/fox-python-test](https://github.com/gregsandell/fox-python-test)

## The instructions were:
Instructions for Candidate
* Write clear, idiomatic Python with type hints where reasonable.- Add brief comments or
  docstrings to explain non-obvious logic.
* Use Dummy Data or APIs
* Aim to complete the tasks in 60–90 minutes.
1. Task 1 – String & Collections
  * Implement most_common_word(text: str, stopwords: set[str] | None = None) -> str | None.
2. Task 2 – Data Structures & Algorithms
  * Implement merge_intervals(intervals: list[list[int]]) -> list[list[int]].
3. Task 3 – OOP Design & API Thinking
  * Create a Logger class with log, get_logs, and search using a dummy API or Open API.
4. Task 4 – Debugging & Refactoring
  * Fix fib function and provide iterative and memoized versions.
5. Task 5 – Hashing & Grouping
  * Implement group_anagrams(words: list[str]) -> list[list[str]].
